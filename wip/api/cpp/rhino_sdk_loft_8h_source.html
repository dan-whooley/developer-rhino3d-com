265 /*
266 Description:
267  Creates a SubD surface lofted through shape curves.
268 Parameters:
269  shape_curves [in] - Array of SubD-friendly NURBS curves to loft through
270  bClosedLoft [in] - Creates a SubD that is closed in the lofting direction. Must have three or more shape curves.
271  bSubDCorners [in] - With open curves, adds creased vertices to the SubD at both ends of the first and last curves.
272  bSubDCreases [in] - With kinked curves, adds creased edges to the SubD along the kinks.
273  nDivisions [in] - The segment number between adjacent input curves.
274  result_subd [in/out] - Pointer to an empty SubD to return the loft result.
275 Returns:
276  true if successful, false otherwise.
277 Remarks:
278  Curves must be in the proper order and orientation and have point counts for the desired surface.
279  Curves must be either all open or all closed.
280 */
