Detailed Description

Contains a simple doubledouble class that performs ONLY multiplication and subtraction Subset of implementation of "Library for Double-Double and Quad-Double Arithmetic" paper and on "Quad Double computation package" by Yozo Hida, Xiaoye S. Li, and David H. Bailey
